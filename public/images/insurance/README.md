# Insurance Logos

This directory should contain logo images for the insurance providers listed in `src/data/Insurance.json`.

## Required Logo Files

- `admedika.png` - AdMedika
- `medika-plaza.png` - Medika Plaza
- `meditap.png` - Meditap

## Image Specifications

- **Format**: PNG (with transparency preferred) or SVG
- **Recommended Size**: 200x100px or similar aspect ratio
- **Background**: Transparent or white
- **Quality**: High resolution for retina displays

## Fallback Behavior

If a logo image is missing, the component will display the insurance provider's name as text fallback.
